PH Printers

Overview

This section includes information on PH Printer configurations, setup, and troubleshooting.

PH Initial Install

  • Download Vega
  • Download GDC Server - Service Installer
  • Change security on
    • C:pulsar
    • C:\scaps
    • C:\program files\fa-tech
    • General data server folder
  • Copy DirectMark files and server setting to General Data Server folder
  • configure the server settings to look for their polling path; ServerSettings.xml tells the server which processor (directmark) to use
  • Copy Sjf file C:\Pulsar\cfg
  • If there is no backup; go to printer serial and download necessary files

Troubleshooting

  • Make sure the printer is connected
  • Check device manager for USB connection and SamLight driver
  • Power cycle the printer and computer
  • Turn on the printer before the computer
